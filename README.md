# Xtramile Java Test

## Task 1
Find numbers that appear only once in the list.

Input example:
[1, 2, 1, 3]

Output:
[2, 3]

## Task 2
Find the largest pair sum in the list.

Input example:
[5, 9, 7, 11]

Output:
20

## How to Run

Run Task 1:

```bash
javac Task1.java
java Task1
```

Run Task 2:

```bash
javac Task2.java
java Task2
```
